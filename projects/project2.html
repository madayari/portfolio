Zarzis Smart Center — VR Tour

Overview

Immersive virtual reality tour of the Zarzis Smart Center, showcasing Zarzis Smart Center building. Users can explore multiple floors, interact with building features, and navigate the architectural space in VR.

Key Features

  • Fully functional elevator system with floor selection and smooth transitions
  • Interactive elements throughout the building including doors, information panels, and environmental objects
  • Graphics optimization focusing on light baking and occlusion culling for smooth VR performance
  • Custom UI system with sprite sheet animations for interactive elements
  • Spatial audio implementation with positional sound effects for immersive presence
  • Architectural visualization with accurate interior design and lighting

Gallery

Technical Details

Built using Unity 2021 LTS with Oculus Integration SDK for Meta Quest 2. Custom C# scripts handle elevator mechanics, floor transitions, interactive object states, and teleportation-based navigation system.

Extensive use of Unity's lighting tools including baked lighting, light probes, and reflection probes to achieve realistic architectural visualization while maintaining optimal VR performance at 72 FPS.
