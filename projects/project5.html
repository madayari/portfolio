Technical Details

Built using Unity 2021 LTS with the XR Interaction Toolkit for VR functionality. Custom C# scripts handle the complex drilling mechanics, equipment interactions, and training assessment logic.

The application supports multiple VR headsets including Meta Quest 2, HTC Vive, and Valve Index. Optimized for smooth performance with high-fidelity graphics to maintain immersion.
